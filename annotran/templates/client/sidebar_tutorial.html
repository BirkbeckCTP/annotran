How to get started

  1. To create a translation in a new language for a selected group, expand a list of existing languages given above by clicking "Select Language" and adding a new language to a language list. Then, follow the instructions in the "Translation Options" below.

  2. To view available translations or edit your own translations select a group and a language above, and follow the instructions in the "Translation Options" below.

  3. To add your own translation into the list of available translations for a selected group and a language above, follow the instructions in the "Translation Options" below.

  4. When viewing a translation, hover over text to see the original version.
