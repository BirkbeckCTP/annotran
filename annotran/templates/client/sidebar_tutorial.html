How to get started

  1. To view an existing translation, select a group and language above.

  2. To create a new translation in an existing language, select a group and language and click "New Translation".

  3. To create a new translation in a new language, select a group and then click "New Translation" in the language list.

  4. To move through the document sentence-by-sentence when writing a translation, click the "lock" button when selecting text.

  5. When viewing a translation, hover over text to see the original version.
